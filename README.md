## Configuration
- ##### To deploy in Firebase

  - Delete the folder /dist (if exists)
  - Open console and type:

  ```bash
  npm run build
  npm run firebase deploy
  ```

- ##### Check package and update

  ```bash
    npm outdated
    npm update
  ```
